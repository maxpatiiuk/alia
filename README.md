# dgs

The ultimate Drewgon compiler

> NOTE: at present this project only includes a parser

## Installation

Install dependencies: 

```sh
make
```

## Running

To see available options, run the script with `--help` argument:

```
./dgc --help
```

Example call:

```sh
./dgc <infile.dg> -t <tokens.txt> 2> <errors.txt>
```
